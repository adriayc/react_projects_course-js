* Inicializar la App React
  - Crear la app
    $ npm create vite@latest react-advanced -- --template react
    $ cd react-advanced
    $ code .          // Open VSCode
    $ npm install
    $ npm run dev
    > Open URL "http://localhost:5173/" in the browser

  - Create otra app (Global Context)
    $ npm create vite@latest global-context -- --template react
    $ cd global-context
    $ code .          // Open VSCode
    $ npm install && npm run dev
    > Open URL "http://localhost:5173/" in the browser
