* Axios HTTP Library
  Axios (URL: https://axios-http.com/) is a primse-based HTTP Client for node.js and the browser. It is isomorphic (= it can run in the browser and nodejs with the same codebase). On the server-side it uses the native node.js module, while on the client (browser) it uses XMLHttpRequests.
