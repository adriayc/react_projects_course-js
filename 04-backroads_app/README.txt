* Inicializar la App React
  - Crear la app
    $ npx create-react-app backroads-app
    $ npx create-react-app@latest backroads-app
    $ cd first-app
    $ code .          // Open VSCode
    $ npm start
    > Open de URL in the browser: 'http://localhost:3000/'

  - Shortcuts in VSCode
    + CTRL + D          // Add selection to next find match
    + SHILT + CTRL + L  // Add additional cursor to all occurrences of the current selection
