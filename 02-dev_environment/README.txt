* Inicializar la App React
  - Crear la app
    $ npx create-react-app first-app
    $ npx create-react-app@latest first-app
    $ cd first-app
    $ code .          // Open VSCode
    $ npm start
    > Open de URL in the browser: 'http://localhost:3000/'

  - Shortcuts in VSCode
    + CTRL + `        // Open terminal
