* Inicializar la App React
  - Crear la app
    $ npx create-react-app react-fundamentals-app
    $ cd react-fundamentals-app
    $ code .          // Open VSCode
    $ npm start
    > Open de URL in the browser: 'http://localhost:3000/'

  - Shortcuts in VSCode
    + CTRL + C    // Stop the server
    + npm start   // Start the server
    + ALT + Click   // Select multiple items
